What's Inside
Core Features
Everything you need to manage a modern library — built in, no plugins, no backend.
📚
50 Books Pre-loaded
Full catalog with 50 tech books across 13 categories — JavaScript, Python, ML, DevOps, Security, Architecture and more.
👥
20 Members Ready
Student, Faculty, Staff, and Premium member types with full profiles including contact, department, and roll number.
📤
Issue & Return System
Complete book issuing with configurable loan periods (7/14/21/30 days) and one-click return with condition tracking.
💰
Auto Fine Calculation
₹10/day overdue fine calculated automatically. Live preview on return screen before confirmation.
⭐
Ratings & Wishlist
5-star rating system for books and personal wishlist. Top-rated books shown on dashboard.
🔐
Role-Based Access
Three roles — Admin (full access), Librarian (manage books/members), Member (browse + wishlist).
📊
Live Analytics Dashboard
Category breakdown chart, overdue alerts, recent activity log, and top-rated books panel.
⚠
Overdue Tracking
Real-time overdue detection with notification bell, dedicated overdue page, and red alert highlights.
💾
Auto Local Save
All data saved to localStorage automatically. Your data persists between sessions — no login to database required.
📥
CSV Export
Export all borrow records as CSV file for reporting, accounting, or external analysis.
🌙
Dark / Light Theme
Toggle between dark and light mode from any screen. Preference saved automatically.
🗄️
Book & Member Database
Full CRUD for books and members from the homepage. Add, edit, delete with search and filter.
Workflow
How It Works
Simple 5-step flow to manage your library efficiently.
01
🔐
Login
Choose your role (Admin/Librarian/Member) and sign in with demo or custom credentials.
02
📚
Manage Books
Browse 50 pre-loaded books. Add new ones, edit details, set availability and copies.
03
👥
Manage Members
View all 20 members. Add students, faculty, or staff with full contact details.
04
📤
Issue / Return
Issue books to members with due dates. Return them with condition and auto fine calculation.
05
📊
Track & Report
Monitor overdue books, view analytics, export CSV, and keep activity logs.

Backend API
REST API Endpoints
Start python3 app.py to enable backend mode. Status: Checking...
🔐
POST /api/auth/login
Login with email + password. Returns JWT token for all API calls.
📚
GET /api/books
List all books. Filter by ?q=, ?category=, ?status=, ?sort=
👥
GET /api/members
List all members. Filter by ?q=, ?type=Student/Faculty etc.
📤
POST /api/borrows/issue
Issue a book. Body: {book_id, member_id, loan_days, issue_date}
📥
POST /api/borrows/:id/return
Return a book. Auto-calculates fine at ₹10/day overdue.
📊
GET /api/dashboard
Full stats: books, members, borrowed, overdue, categories, top-rated.
💰
POST /api/fine/calculate
Calculate fine. Body: {issue_date, return_date, loan_days, rate}
📥
GET /api/export/borrows
Download all borrow records as CSV file.
